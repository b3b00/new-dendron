# repository selector

the repositories.svelet componen allow to load and list all available user github repo.
A field is used to filter repos.

the goal is to be able to navigate with arrows (up and down). 
A user can filter and then navigate to select the desired repo going up and down.
the currently selected repo in the list should be highlighted 
ideally user should be able to continue filter after having starting navigating with arrows
When a repo is selected hitting enter key validate the choice.

User still should be able to use mouse.
