# Command Palette

The Command Palette provides a centralized interface for advanced actions and configuration within the Dendr-Online application. It is accessible globally via a keyboard shortcut.

## Features

- **Search in Notes**: Allows users to search within the content of all notes.
- **Global Access**: Accessible from anywhere in the application (Stash and Tree views) using a keyboard shortcut.
- **Command Palette Integration**: Includes options to toggle search-in-content functionality from the command palette.
